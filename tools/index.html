# git sshkey

  1. 生成新的 sshkey
   ssh-keygen -t rsa -b 4096 -C "你的邮箱地址"
  1. 如果不使用密码连输按 3 次回车
// 保存位置
> Enter a file in which to save the key (/Users/you/.ssh/id_rsa): [Press enter]
// 输入密码
> Enter passphrase (empty for no passphrase): [Type a passphrase]
// 重复密码
> Enter same passphrase again: [Type passphrase again]
  1. 复制 sshkey
windows 文件位置 C:\Users\Administrator\.ssh 命令行 cd ~/.ssh 找到 id_rsa.pub
文件，打开文件进行复制
  1. 登录 github 点击头像->setting 左侧 SSH and GPP KEYS
点击 new sshkey title 自己随便填 sshkey 粘贴复制的密钥
